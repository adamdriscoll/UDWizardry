# UDWizardry

This is a powershell module which has been based upon the following react component:-
[!ref Original Component](https://www.npmjs.com/package/react-wizardry)

## Demo Dashboard
```
New-UDDashboard -Title 'UDWizardry' -Content {
New-UDRow -Columns {
  New-UDColumn -Content {
New-UDWizardry -BodyHeight 350 -Pages {
  [PSCustomObject]@{
    title = "Name"
    fields = @([ordered]@{
      label = "First Name"
      name = "fname"
      type = "text"
      isRequired = $true
    },
    [ordered]@{
      label = "Last Name"
      name = "lname"
      type = "text"
      isRequired = $false
    })
  },[PSCustomObject]@{
      title = "Second"
      fields = @([ordered]@{
        label = "Email Address"
        name = "eaddress"
        type = "email"
        isRequired = $false
      }
      )
    } | ConvertTo-Json -AsArray -Depth 4 | ConvertFrom-Json
  }
  }
}
}
```
